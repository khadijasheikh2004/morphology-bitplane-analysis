# Morphological Processing and Bit-Plane Analysis in MATLAB

## Overview
This project explores morphological operations and bit-plane slicing for structural image analysis.

---

## Why I Built This
This project helped me understand:
- Shape-based image processing
- Binary image manipulation
- How image information is distributed across bit levels

---

## Part A: Morphological Processing
- Image binarization
- Opening (Erosion → Dilation)
- Closing (Dilation → Erosion)
- Boundary extraction using subtraction

---

## Part B: Bit-Plane Slicing
- Extraction of all 8 bit planes
- Analysis of fine vs coarse details
- Custom morphological enhancement on each plane

---

## Features
- Step-by-step visualization
- Saves all intermediate outputs
- Structural enhancement techniques
